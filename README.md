# 🚚 Yagi Logistics - Real-Time Parcel Tracking System

A clean, modern, and fully functional **parcel tracking web application** with an admin dashboard and customer tracking portal. Built with Firebase for real-time updates.

---

## ✨ Key Features

### 🔐 Admin Dashboard
- Create new parcels with auto-generated or custom Tracking IDs
- Update parcel status in real-time (Pending → In Transit → Delivered)
- Delete parcels
- Automatic delivery timeline recording

### 📦 Customer Tracking
- Track parcels using Tracking ID
- View delivery timeline history
- Estimated delivery countdown
- Shareable tracking links (`?track=YAG123456`)

### ⚡ Real-time Updates
- Live synchronization using Firebase Realtime Database
- Instant updates across all devices

### 📱 Responsive UI
- Mobile-friendly design
- Smooth animations
- Clean glassmorphism interface

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Firebase Realtime Database
- **Auth:** Firebase Authentication (optional upgrade)
- **Design:** Modern UI (glassmorphism + gradients)

## 🚀 How It Works

1. Admin logs in and creates a parcel
2. System generates a tracking ID
3. Customer enters ID or opens tracking link
4. Admin updates status → Customer sees updates instantly
